# react-test-task

### This project is a test task using the following technologies:###

Front-end:

+ React.js;
+ Redux;
+ 2gis api;
+ Less;
+ Webpack.

Back-end:

+ Node.js + express;
+ MongoDB + Mongoose.


### Production commands: ### 

+ Install packages 

    >yarn install

+ Build project

    >yarn production

+ Start node server

    >node server.js

### Development commands: ###

+ Build project to develop
                
    >yarn build

+ Start dev-server

    >yarn start
